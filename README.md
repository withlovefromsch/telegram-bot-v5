# 🎓 Курсы на миллион — Telegram Bot V5

Telegram-бот для продажи онлайн-курсов с полным циклом: регистрация → каталог → корзина → заявка → оплата → доступ к курсу.

---

## 📁 Структура проекта

```
telegram_bot_v5/
├── bot.py                  # Точка входа
├── config.py               # Настройки (env-переменные)
├── .env.example            # Пример файла окружения
├── requirements.txt        # Зависимости
│
├── data/
│   └── courses.py          # База курсов
│
├── database/
│   └── db.py               # Все операции с SQLite
│
├── states/
│   └── states.py           # FSM-состояния
│
├── keyboards/
│   ├── reply.py            # Reply-клавиатуры
│   └── inline.py           # Inline-клавиатуры
│
├── filters/
│   └── is_admin.py         # Фильтр для администраторов
│
└── handlers/
    ├── registration.py     # Регистрация пользователя
    ├── profile.py          # Личный кабинет
    ├── courses.py          # Каталог курсов + корзина
    ├── orders.py           # Заявки (сторона клиента)
    ├── help.py             # Помощь / поддержка
    └── admin.py            # Администраторская панель
```

## 🚀 Установка и запуск

### 1. Клонировать / скачать проект

```bash
cd telegram_bot_v5
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate   # Linux/macOS
# или
venv\Scripts\activate      # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

```bash
cp .env.example .env
```

Заполните `.env`:

| Переменная        | Описание                                  |
|-------------------|-------------------------------------------|
| `BOT_TOKEN`       | Токен бота от @BotFather                  |
| `ADMIN_IDS`       | Telegram ID администраторов через запятую |
| `SUPPORT_USERNAME`| Username техподдержки (с @)              |

### 5. Запустить бота

```bash
python bot.py
```

## 🔄 Логика бота

### Клиент
```
/start → Регистрация (Имя → Телефон → Email)
       → Главное меню

Главное меню:
├── 👤 Личный кабинет  — данные профиля + список заявок
├── 📚 Список курсов   — каталог с добавлением в корзину
├── 🔓 Открытые доступы — купленные курсы со ссылками
└── ❓ Помощь           — контакт техподдержки

Покупка курсов:
Каталог → Корзина → Оформить заявку → Подтвердить
→ Ожидание ссылки на оплату от менеджера
→ Оплата → Скриншот чека → [Оплачено]
→ Ожидание открытия доступа
→ Уведомление с ссылкой на курс
```

### Администратор
```
Получает заявку → [В работе] → вводит ссылку на оплату
Получает скриншот чека → [В работе] → проверяет → открывает доступ вручную
→ [Завершить] → клиент получает уведомление с ссылкой

Команды:
/orders — список активных заявок
```

## 🗃️ База данных

SQLite (`bot_database.db`) создаётся автоматически при первом запуске.

Таблицы:
- `users` — зарегистрированные пользователи
- `cart` — корзина покупок
- `orders` — заявки
- `order_items` — позиции в заявке
- `admin_messages` — message_id сообщений администраторов (для редактирования)
- `course_accesses` — выданные доступы к курсам
