# Telegram Bot

## Запуск без виртуального окружения

Этот проект можно запускать без `venv` — достаточно установить зависимости глобально или в пользовательскую директорию и запускать из корня проекта.

### Шаги на сервере

1. Убедитесь, что установлен Python 3:
```bash
python3 --version
```

2. Установите зависимости:
```bash
python3 -m pip install --user -r requirements.txt
```

3. Создайте файл `.env` с токеном и настройками (пример есть в `.env.example`).

4. Запустите бота:
```bash
cd /path/to/project
python3 -m bot
```

### Быстрый запуск

Можно использовать скрипт:
```bash
./run.sh
```

### Примечание

- `python3 -m bot` запускает приложение из корня проекта и гарантирует, что модули из `data/`, `handlers/` и других пакетов будут найдены.
- Если сервер не использует виртуальное окружение, пакеты будут установлены в пользовательскую директорию `~/.local`.
